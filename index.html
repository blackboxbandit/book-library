How to Import Your Amazon Wishlist

A script is included with this app that you paste into your browser's Developer Console while viewing your Amazon wishlist. It extracts all items and downloads a JSON file you can import here.

Step 1: Open Your Amazon Wishlist

  1. Go to amazon.co.uk/hz/wishlist/ls/ (or your regional Amazon)
  2. Scroll all the way down so every item loads (Amazon uses infinite scroll)

Step 2: Open the Developer Console

  1. Chrome / Edge: Press Cmd + Option + J (Mac) or Ctrl + Shift + J (Windows)
  2. Firefox: Press Cmd + Option + K (Mac) or Ctrl + Shift + K (Windows)
  3. Safari: First enable Developer menu: Safari → Settings → Advanced → Show features for web developers. Then press Cmd + Option + C

Step 3: Paste & Run the Script

  1. Open the file tools/amazon-wishlist-exporter.js from the Book Library folder
  2. Copy the entire contents
  3. Paste into the console and press Enter
  4. A JSON file will download automatically

Step 4: Import into Book Library

  1. Come back to this app's Wishlist tab
  2. Click Import Wishlist and select the downloaded JSON file
  3. Your items will be imported with titles, authors, prices, and links
Tip: If Safari blocks the paste, it may ask you to type allow pasting first. This is a security feature — just type it and press Enter, then paste the script.
